# 夏川心爱Cocoa 聊天室

💛🤍 和小天使 Cocoa 聊天吧！

## 使用方法

1. 打开网页
2. 点击右上角 ⚙️ 设置
3. 选择 API 服务商（DeepSeek 或通义千问）
4. 输入你的 API Key
5. 开始聊天！

## API Key 获取

- **DeepSeek**: https://platform.deepseek.com/
- **通义千问**: https://dashscope.console.aliyun.com/

## 注意事项

- API Key 保存在浏览器本地，不会上传到任何服务器
- 请妥善保管你的 API Key
- 这是一个纯前端应用，所有对话直接发送到 LLM API

夏川心爱Cocoa 是 HeartTune 虚拟偶像团体成员，以"小天使"形象进行活动的 VTuber。
